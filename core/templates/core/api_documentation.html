Overview

The backend is organized around users, projects, beneficiaries, donations, and project update subscriptions. Most JSON responses follow the format {"success": true, "message": "...", "data": ...}. Paginated list endpoints usually return count, next, previous, and results inside data.

  • Base API prefix: {{ base_url }}
  • Authentication: Bearer JWT access tokens
  • Refresh endpoint: /api/users/token/refresh/
  • Media files are served from /media/ while DEBUG is enabled

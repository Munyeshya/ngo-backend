Overview

The backend is organized around users, projects, beneficiaries, donations, and project update subscriptions. Most JSON responses use the shape {"success": true, "message": "...", "data": ...}.

  • Base API prefix: {{ base_url }}
  • Authentication: Bearer JWT access tokens
  • Refresh endpoint: /api/users/token/refresh/
  • List endpoints usually return count, next, previous, and results
  • Media files are available under /media/ while debug serving is enabled
